O que é o JavaScript? 🤔

JavaScript é uma linguagem de programação usada para dar interatividade e comportamento às páginas da web. Enquanto o HTML cria a estrutura do site e o CSS cuida da aparência, o JavaScript faz a página reagir às ações do usuário.

Com ele, é possível criar funções como abrir menus, validar formulários, mostrar mensagens, alterar textos e imagens, atualizar conteúdos sem recarregar a página e até desenvolver jogos, aplicativos e sistemas completos.

De forma simples, o JavaScript permite que o site deixe de ser “parado” e passe a ser dinâmico. Por exemplo: quando você clica em um botão e algo acontece na tela, geralmente existe JavaScript por trás disso.

Ele é muito importante no desenvolvimento web porque é uma das principais linguagens usadas na internet. Hoje, além dos sites, também pode ser usado no back-end, em aplicativos para celular e em várias outras áreas da programação.

  • 🔒Constantes

    • Constante é um valor que não muda durante a execução do programa. Ou seja, depois que você define, ele permanece fixo.

  • 📦Variaveis

    • Variáveis são espaços na memória que guardam valores que podem mudar durante a execução de um programa.

  • 🔀Condições

    • Condições são decisões no programa, usadas para executar algo dependendo de uma situação.

  • 📚Vetores

    • Valores são dados armazenados em variáveis ou constantes, como números, textos ou verdadeiro/falso.

  • 🧮Matrizes

    • Matrizes são estruturas de dados organizadas em linhas e colunas.

  • ⚙️Funçoes

    • Funções são blocos de código criados para executar uma tarefa específica.

  • 🔁Repetiçõe

    • Repetições executam uma ação várias vezes enquanto uma condição for verdadeira.

    • 🔙Com Retorno

      • Função que retorna um resultado depois de executar uma tarefa.

    • ▶️Sem Retorno

      • Função que apenas executa uma ação, sem devolver um valor.
