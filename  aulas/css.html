O que é o Css ? 🤔

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência e o estilo de uma página web.

Enquanto o HTML cria a estrutura do conteúdo, o CSS controla como esse conteúdo será exibido, como cores, tamanhos de texto, fontes, espaçamentos, posicionamento e layout dos elementos.

Com o CSS é possível deixar o site mais organizado, bonito e responsivo, adaptando a aparência para diferentes telas como computadores, tablets e celulares.

De forma simples, o CSS é responsável por transformar a estrutura criada pelo HTML em uma interface visual agradável e profissional.
